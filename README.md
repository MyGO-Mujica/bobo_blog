# MyGO 博客客户端

一个基于 Vue 3 + Vite 的现代化博客平台前端项目。

## 🎯 主要功能

- 📝 **文章管理** - 完整的文章增删改查，支持富文本编辑
- 🏷️ **分类管理** - 灵活的文章分类系统
- 👤 **用户认证** - 登录、注册、个人中心
- 🌐 **博客广场** - 多人内容共享平台
- 📱 **响应式设计** - 支持桌面和移动设备

## 🛠 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 下一代前端构建工具
- **Pinia** - 状态管理
- **Vue Router** - 路由管理
- **Axios** - HTTP 客户端
- **Element Plus** - UI 组件库
- **Editor.js** - 富文本编辑器
- **Sass** - CSS 预处理器

## 📁 项目结构

```
src/
├── api/              # API 接口
│   ├── user.js       # 用户 API
│   ├── article.js    # 文章 API
│   └── square.js     # 广场 API
├── components/       # 全局组件
├── router/          # 路由配置
├── stores/          # 状态管理（Pinia）
├── styles/          # 全局样式
├── utils/           # 工具函数
│   ├── request.js   # Axios 配置
│   └── format.js    # 格式化工具
├── views/           # 页面级组件
│   ├── home/        # 首页
│   ├── login/       # 登录
│   ├── user/        # 用户管理
│   ├── article/     # 文章管理
│   └── square/      # 博客广场
└── App.vue          # 根组件
```

## 🚀 快速开始

### 安装依赖
```bash
pnpm install
```

### 开发模式
```bash
pnpm dev
```

### 生产构建
```bash
pnpm build
```

## 📚 主要模块

### 1. 用户模块
- 注册和登录
- 个人资料管理
- 头像修改
- 密码修改
- Token 自动管理

### 2. 文章管理
- 文章列表展示
- 分类管理（增删改查）
- 文章编写和发布
- 图片上传
- 文章搜索和筛选

### 3. 博客广场
- 浏览所有用户发布的文章
- 点赞和评论
- 内容管理员功能

## 🔑 核心功能说明

### API 请求拦截
- 自动添加 Token 认证
- 统一错误处理
- 401 错误自动跳转登录

### 路由权限控制
- 登录状态检查
- 角色权限管理
- 自动重定向

### 状态管理
- 用户信息管理
- 登录状态持久化
- 全局计数器

## 📝 注意事项

- 项目使用 `pnpm` 作为包管理器
- 路由使用懒加载，支持性能优化
- 组件使用 `<script setup>` 现代语法
